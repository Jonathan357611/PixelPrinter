# PixelPrinter

---

Package to show pictures in the Command line.

---

# Usage

Installation
````shell
git clone https://github.com/Jonathan357611/PixelPrinter
````

Usage
````shell
python3 -m PixelPrinter.py [-i/--image <filename>] [-d/--downscale <Downscale-Factor> (Optional)]
````
